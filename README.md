# 🤖 AI Agent Automation Project

An intelligent AI Agent built using **n8n**, **OpenAI GPT Model**, **MongoDB Memory**, **Webhook Automation**, and **Web Search Tools**.

This AI Agent can understand user messages, search live information, remember conversations, and automatically respond through webhook integrations.

---

# 🚀 Features

- 💬 Smart AI Chat Response
- 🌐 Live Web Search Support
- 🧠 Chat Memory using MongoDB
- 🔗 Webhook Integration
- ⚡ Automated Responses
- 🌍 Hindi + English Support
- 🤖 Browser Automation Support
- 📚 Context-Aware Conversations

---

# 🛠️ Tech Stack

- **n8n**
- **OpenAI API**
- **MongoDB**
- **Webhook**
- **Browserbase**
- **HTTP Request Tool**

---

# 📌 Workflow Overview

Webhook → AI Agent → OpenAI Chat Model
                         ↓
                   MongoDB Memory
                         ↓
                    Web Search Tool
                         ↓
                Respond to Webhook

✅ Answer user questions
